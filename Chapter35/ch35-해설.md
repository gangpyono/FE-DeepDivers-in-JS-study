# Chapter35

## 📌문제1. 출력결과물을 예측하세요.

### 문제 작성

```js
const obj1 = { a: 10, b: 2 };
const obj2 = { c: 3, a: 1 };
const merged = { ...obj1, ...obj2 };
console.log(merged); // (1)
```

### 정답 및 해설

```
(1): {a: 1, b:2, c:3}
일반 객체가 이터러블이 아니지만, 객체 리터럴 내부에선 스프레드 문법을 사용할 수 있다.
프로퍼티가 중복되는경우 뒤에 위치한 프로퍼티가 우선권을 갖는다.
```
