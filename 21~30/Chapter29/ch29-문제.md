## 📌문제1

### 코드 실행 결과를 예측히세요.

```js
Math.abs(); // (1)
Math.abs(null); // (2)
Math.abs(undefined); // (3)
Math.abs(""); // (4)
Math.abs("-1"); //(5)
```

### 답안 작성

```
(1) : 
(2) : 
(3) : 
(4) : 
(5) : 
```
