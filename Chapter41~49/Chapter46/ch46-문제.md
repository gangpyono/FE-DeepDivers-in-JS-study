## 📌문제1
### 다음 코드의 실행 결과를 쓰시오
```js
function* genFunc(){
  const x = yield 1;
  const y = yield(x + 1);
  return x + y;
}

const generator = genFunc(0);

let res = generator.next(1);
console.log(res);

res = generator.next(2);
console.log(res);

res = generator.next(10);
console.log(res);
```

## 📌문제2
### 다음 코드의 실행 결과를 쓰시오
```js
    function* generater1() {
        yield 1;
        yield 2;
        yield 3;
        return 4;
        yield 5;
      }

      const f1 = generater1();
      console.log(f1.next()); // (1)

      for (const value of f1) {
        console.log(value); // (2)
      }

      console.log([...f1]); // (3)
```

### 답안 작성
```
(1)
(2)
(3)
```
