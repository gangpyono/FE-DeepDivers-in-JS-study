## 📌문제2
### 출력 결과물을 예측해 주세요.
```js
   const s1 = Symbol("m1");
   const s2 = Symbol("m1");

   console.log(s1 === s2); // (1)

   const s3 = Symbol.for("mySymbol");
   const s4 = Symbol.for("mySymbol");

   console.log(s3 === s4); // (2)
```
### 답안 작성
```
(1)
(2)
```
