# Chapter17 생성자 함수에 의한 객체 생성
## 📌문제1
### 문제 작성
```js
function Func(){
  this.a = 100;
  this.getA = () => { return this.a };
  return 100;
}

const func1 = Func();
const func2 = new Func();
console.log(func1);
console.log(func2);

```
### 정답 및 해설
```
100
Func {a: 100, getA: ƒ}
func1은 일반 함수로서, func2는 생성자로서 Func 가 호출되었기 때문이다.
```
