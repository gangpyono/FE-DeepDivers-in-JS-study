## 📌문제2
### 다음 코드의 출력결과를 예측해주세요.
```js
const a = undefined;
const b = null;
const c = 123;


const a1 = a ?? "값이 없습니다.";
console.log(a1); //(1)


const b1 = b ?? "값이 없습니다.";
console.log(b1); //(2)


const c1 = c ?? "값이 없습니다.";
console.log(c1); //(3)	
```
### 답안 작성
```
(1)
(2)
(3)
```
