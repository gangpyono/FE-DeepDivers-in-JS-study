# Chapter08
## 📌문제1
### 1. if~else구문과 switch구문의 차이점에대해서 설명하세요.

### 답안 작성
```
if~else문은 조건문이 불리언값으로 평가되어야 하지만, switch구문은 불리언,문자열,숫자로 평가될 수 있다.
```

## 📌문제2
### 다음 코드의 실행 결과를 쓰시오
```js
let x = 0;
switch (x){
case 0 : x++;
case 1 : x += 2;
case 2 : x += 3;
default : x += 4;
}
console.log(x);
```
### 정답 및 해설
```
10
break 문이 없어서 차례로 전부 실행된다.
```
